Why PotatoOS?

PotatoOS is a "parasite" operating system designed to work on a host OS like Windows or Linux instead of bare-metal.

Runs in your desktop

There's no need to install any complex bootloader or deal with partitions, PotatoOS runs directly on your existing OS.

STARCH scripting language

PotatoOS's applications are built using the interpreted language STARCH, which has a simple syntax with easy system APIs.

Cross-platform

Because PotatoOS is entirely coded and built with Godot Engine, it's completely compatible with all platforms.

Lightweight and portable

No complex virtual machines, no bloated installs (cough cough Microsoft). PotatoOS fits in a single directory and can be moved between systems easily.

Familiar yet unique

A Unix-like filesystem and Bash mirror provide a comfortable environment, while features like the STARCH pipeline operator bring fresh ideas to scripting.
